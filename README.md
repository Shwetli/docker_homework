# 🔒 Chat Wall

Full-stack Docker проект с:

- 🐍 Flask backend (Python)
- 💾 PostgreSQL база
- 🌐 HTML/JS frontend
- 🐳 Docker Compose

## ▶️ Стартиране

```bash
docker compose up --build
```

## 📦 Функции

- Въвеждаш съобщение
- Записва се в базата
- Показва се в таблица отдолу

## 🔗 Достъп

- Frontend: http://localhost:8080
- Backend API: http://localhost:5000/api/messages
